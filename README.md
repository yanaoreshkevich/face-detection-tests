# Face SDK Automation Tests

This project demonstrates UI and API testing for Face SDK Web application.

## Stack
- Cypress
- JavaScript
- Page Object Model (POM)

## Tests

### UI Test

- Navigate to Face Detection tab
- Upload image (.jpg)
- Verify that preview image is updated

### API Test

- Send face detection request
- Validate response status and body

## Installation

npm install

## Run tests

npx cypress open
